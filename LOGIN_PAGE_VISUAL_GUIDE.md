## 🎭 Анимации:

### Blob анимация:
```css
@keyframes blob {
  0%, 100% { transform: translate(0, 0) scale(1); }
  25%      { transform: translate(20px, -50px) scale(1.1); }
  50%      { transform: translate(-20px, 20px) scale(0.9); }
  75%      { transform: translate(50px, 50px) scale(1.05); }
}

duration: 7s
timing: infinite
delays: 0s, 2s, 4s
```

### Кнопка:
```css
hover: scale(1.02)
active: scale(0.98)
transition: all 0.2s ease
```

### Поля ввода:
```css
focus: ring-2 ring-orange-500
transition: all 0.3s ease
```

## 📱 Адаптивность:

```
Mobile (< 640px):
  - Карточка: padding 8 (2rem)
  - Текст: стандартный размер
  - Все элементы вертикально

Tablet (640px - 768px):
  - Карточка: padding 10 (2.5rem)
  - Увеличенный spacing

Desktop (> 768px):
  - Карточка: padding 10 (2.5rem)
  - Максимальная ширина: 28rem (448px)
  - Центрирование по вертикали и горизонтали
```

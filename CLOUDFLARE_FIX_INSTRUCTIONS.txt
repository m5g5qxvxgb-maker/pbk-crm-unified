🔥 СРОЧНО: CLOUDFLARE БЛОКИРУЕТ ДОСТУП К САЙТУ

═══════════════════════════════════════════════════

❌ ПРОБЛЕМА:
https://crmpbk.appp2p.uk возвращает ошибку 403 Forbidden
Cloudflare Firewall блокирует все запросы

═══════════════════════════════════════════════════

✅ РЕШЕНИЕ (ШАГИ):

1️⃣ ЗАЙТИ В CLOUDFLARE
   → https://dash.cloudflare.com
   → Выбрать домен: appp2p.uk

2️⃣ ОТКЛЮЧИТЬ "UNDER ATTACK MODE"
   → Security → Settings
   → Security Level → Найти "Under Attack Mode"
   → Переключить на "Medium" или "Low"

3️⃣ ПРОВЕРИТЬ FIREWALL RULES
   → Security → WAF → Firewall Rules
   → Если есть правила блокирующие crmpbk - удалить
   
4️⃣ СОЗДАТЬ ПРАВИЛО BYPASS (опционально)
   → Firewall Rules → Create a Firewall Rule
   → Название: "CRM Bypass"
   → Field: Hostname
   → Operator: equals
   → Value: crmpbk.appp2p.uk
   → Then: Allow

5️⃣ ОЧИСТИТЬ КЭШ
   → Caching → Configuration
   → Purge Everything
   → Подтвердить

6️⃣ ПРОВЕРИТЬ SSL MODE
   → SSL/TLS → Overview
   → Должно быть: "Full" или "Full (Strict)"

# 🌐 Website Module - Agent Instructions

## 👤 Роль
Агент разработки **Website модуля** - публичный сайт компании PBK Construction.

## 📋 Компоненты (Требует редизайна)
- Главная страница
- О компании
- Портфолио проектов
- Услуги
- Форма заявки
- Контакты
- Блог (опционально)

## 📁 Файлы (Будут созданы)
```
/root/pbk-website/
├── pages/
│   ├── index.tsx           # Главная
│   ├── about.tsx          # О компании
│   ├── portfolio.tsx      # Портфолио
│   ├── services.tsx       # Услуги
│   └── contact.tsx        # Контакты
├── components/
│   ├── Hero.tsx           # Hero section
│   ├── LeadForm.tsx       # Форма заявки
│   └── ProjectCard.tsx    # Карточка проекта
└── public/
    └── images/            # Фото проектов
```

## 🎯 Технологии
- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Forms:** React Hook Form
- **SEO:** Next.js metadata API
- **Analytics:** Google Analytics

## 🔥 Задачи
1. ⏳ Современный дизайн (UI/UX)
2. ⏳ Мобильная версия (responsive)
3. ⏳ Форма заявки → CRM webhook
4. ⏳ Портфолио с фото проектов
5. ⏳ SEO оптимизация
6. ⏳ Multilingual (PL/EN/RU)
7. ⏳ Contact form → Email service
